# Jana Nayagan (Chennai) BookMyShow Notifier

This script monitors BookMyShow for new theatres showing **Jana Nayagan** in **Chennai**.

## Notification Topic

Push notifications are sent to the following **ntfy.sh** topic:
* **Topic**: `jana-nayagan-chn-z4b7k9`
* **Subscription Link**: [https://ntfy.sh/jana-nayagan-chn-z4b7k9](https://ntfy.sh/jana-nayagan-chn-z4b7k9)
